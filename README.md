## Installation

1. Clone the repository
```bash
git clone https://github.com/pritamsaha07/email_auto_reminder_.git
```

2. Install dependencies for backend 
```bash
cd server
npm install
```
3. Start the server
```bash
npm run dev
```
4.Install frontend dependencies
```bash
cd ../client/myapp
npm install
```
Start React App
```bash
npm start
```
